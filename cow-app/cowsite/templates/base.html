Welcome to Cowsite!

This is the web page access point for accessing the database system for the cow data. Here you can insert data, fetch/download data and view overall meta data. To learn more, please keep reading.


Insert/Scan for data files

To insert files into the database, go to the File Scan tab.


Query/Download data

To download data from the database, go to the Swedish data tab to access data from the Swedish farm or go to the tab Dutch data to access data from the Dutch farm.


Overview - Meta data

To see meta data regarding the content of the database system, go to the Overview tab.
