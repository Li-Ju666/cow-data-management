Welcome to Cowsite!


File scan

To upload files to the database, the file(s) is copied to the folder <project directory>/input_files. Position data is copied to the folder called input_files/pos and all other files are copied to the folder called input_files/info. When a copy of the file(s) is placed in the correct folder, go to the upload tab and press scan folder. Files that have not previously been uploaded, will then be inserted to the database. Note that uploading position data can take up to 30 minutes per file. When a file is done being uploaded, it can be removed from the folder. Be careful not to remove or alter the log file.


Fetching/Quering data

To query information from the database, go to the Swedish Data tab and select what type of data is requested. Then, one must select what cows are of interest. The user must then select a requested time interval. The queried data will be written to one or more created files within the uploading folder in <project directory>/result_files.


Overview

The Overview tab displays basic overview of the database regarding number of records and storage used. Also inserted files of the logs are shown.
